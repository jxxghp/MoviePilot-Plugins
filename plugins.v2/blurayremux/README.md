# 蓝光原盘转封装

整理电影蓝光原盘目录时，使用 `ffmpeg` 将 BDMV 分片转封装为单个 MKV 文件。

## 功能边界

- 默认关闭，需要在插件配置中启用。
- 仅处理本地源到本地目标。
- 仅处理电影类型的 BDMV 目录。
- 仅支持 `copy` 和 `move` 整理方式。
- 未命中处理条件时返回原生整理流程。

## 转封装规则

- 优先解析 `BDMV/PLAYLIST/*.mpls`，选择匹配到的最大播放列表。
- 支持按 MPLS 时间戳写入 `ffconcat` 裁剪点。
- 若存在播放列表但无法匹配分片，则不回退到最大文件。
- 若无播放列表且存在多个 m2ts 分片，则不回退到最大文件。
- `move` 模式只允许从蓝光根目录入口整理，避免误删子目录入口的源目录。
