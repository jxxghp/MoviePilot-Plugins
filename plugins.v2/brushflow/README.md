# 站点刷流

站点刷流 V5 使用 Vue 模块联邦提供任务工作台。每项任务独立绑定一个站点和一个下载器，并拥有自己的调度周期、选种与删种规则、托管记录、统计和运行诊断。

## 升级兼容

- V5 依赖 MoviePilot `v2.14.6+` 提供的 Vue 联邦宿主能力，旧版宿主会在安装和更新前拒绝该版本。
- 首次加载 V5 时，旧版 `brushsites` 会按站点拆分为独立任务。
- 旧版站点覆盖 JSON 会合并到对应任务中。
- 原有种子、归档和未托管数据会按站点迁移到任务命名空间。
- 已有的全局 `刷流` 标签继续识别；V5 新增任务唯一标签，避免多个任务互相接管。

## 前端入口

插件构建产物暴露以下联邦组件：

| 组件 | 用途 |
| --- | --- |
| `AppPage` | 主界面侧栏中的完整刷流工作台 |
| `Page` | 插件详情弹窗 |
| `Config` | 插件配置弹窗 |
| `Dashboard` | 仪表板任务概览 |

侧栏入口由插件的 `get_sidebar_nav()` 注册，可在工作台全局设置中关闭。

## 插件 API

所有接口均使用宿主 Bearer 认证，基础路径为 `/api/v1/plugin/BrushFlow`。

| 方法 | 路径 | 用途 |
| --- | --- | --- |
| `GET` | `/status` | 获取全局状态、任务摘要与可选站点下载器 |
| `POST` | `/settings` | 更新插件与侧栏入口开关 |
| `POST` | `/tasks` | 创建任务 |
| `GET` / `PUT` / `DELETE` | `/tasks/{task_id}` | 查询、更新或删除任务 |
| `POST` | `/tasks/{task_id}/state` | 启用或暂停任务 |
| `POST` | `/tasks/{task_id}/run` | 立即执行刷流刷新 |
| `POST` | `/tasks/{task_id}/check` | 立即执行种子检查 |
| `POST` | `/tasks/{task_id}/clear` | 清除当前任务的独立数据 |

## 构建

在本目录安装依赖后执行 `yarn build`，产物生成到 `dist/assets`，由插件后端的 Vue 渲染模式加载。
