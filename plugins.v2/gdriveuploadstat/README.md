# 网盘上传量统计

按盘符统计 MoviePilot 整理记录（TransferHistory）的落盘数据量，达到自定义阈值时通过通知频道告警。自用于 gclone 多盘上传限额监控。

## 原理

整理记录里每条成功入库都带有目标文件项 `dest_fileitem`（含 `size` 字节数）和目标路径 `dest`。盘符（如 `LK01`）藏在路径段里（`/fufu/LK01/...`）。插件按你配置的盘符关键字对目标路径做路径段匹配，把命中记录的 `size` 求和，得到该盘在统计窗口内的上传量。

## 统计口径

- 仅计入 `status=成功` 的记录
- 仅计入 `move`（移动）/ `copy`（复制）模式，`link` 等不产生实际上传的模式不计
- 单条大小取 `dest_fileitem.size`，缺失时降级取 `src_fileitem.size`，再缺失记为"未计量"并在通知中提示
- 换算按 1024 进制（GiB），与 MP 界面显示一致

## 配置项格式

### 盘符规则（每行一个）

格式：`显示名,盘符,阈值GB`

```
LK01盘,LK01,700
LK02盘,LK02,2000
Gd01盘,Gd01,2000
```

- 盘符：原样填路径里那一段，区分大小写，不加斜杠。插件内部按 `/盘符/` 路径段匹配
- 显示名：可留空，留空则用盘符本身
- 阈值：纯数字，单位 GB(1024)，无上限。留空或 0 = 该盘不告警
- 支持中文逗号；`#` 开头的行为注释

### 统计模式

- `自然日`（推荐）：统计当前自然日从"起算时间"起的累计量
- `滚动近N小时`：只看最近 N 小时

### 自然日起算时间

24 小时制 `HH:MM`，如 `00:00`（本地零点起算）、`15:00`。填 `15:00` = 每天 15:00 到次日 15:00 算一天。时区跟随 MP 容器时区。

### 检查频率（cron）

标准 5 段 `分 时 日 月 周`，如 `*/30 * * * *`（每 30 分钟）、`0 * * * *`（每小时整点）。

## 阈值告警

开启"多级预警"时，每盘每日在 80% / 95% / 100% 三档各告警一次（同档不重复）。关闭则只在 100% 告警一次。告警去重记录保留最近 7 天。
